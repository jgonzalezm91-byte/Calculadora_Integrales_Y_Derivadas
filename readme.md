 Descripción del proyecto

Esta página web es una calculadora interactiva diseñada para realizar operaciones de Cálculo Diferencial e Integral de forma sencilla y visualmente atractiva.  
El diseño está construido con Tailwind CSS para mantener un estilo moderno y responsivo, además de incluir un fondo animado tipo galaxia con partículas suaves.



 Funcionalidades principales

1. Derivación simbólica  
   Permite obtener la derivada simbólica de una función introducida por el usuario, utilizando la librería `math.js`.

2. Integración numérica:  
   Calcula la integral definida de una función dentro de un intervalo `[a, b]`, empleando la **Regla del Trapecio** con 10,000 subdivisiones para obtener una aproximación precisa.

3. Teclado rápido  
   Incluye botones con funciones comunes (sen, cos, log, etc.), números y operadores para facilitar la escritura de expresiones matemáticas.

4. Interfaz visual:
   - Fondo animado con partículas tipo estrellas (efecto galaxia).  
   - Diseño adaptativo y limpio, ideal para pantallas grandes o móviles.  
   - Campo de resultados visible y secciones bien separadas para cada tipo de cálculo.



 Tecnologías utilizadas

- HTML5  
- CSS3 (Tailwind CDN)  
- JavaScript  
- Math.js (12.4.0) para cálculos matemáticos.  



 Uso básico

1. Escribe la función en el campo `Función f(x)` (por ejemplo: `x^3 + cos(x)` o `1/x`).  
2. Define la variable (por defecto es `x`).  
3. Para derivar, presiona "Derivar Función".  
4. Para integrar, ingresa los límites `a` y `b` y presiona **"Calcular Integral Definida".  
5. El resultado aparecerá en la sección inferior.



## Nota sobre los resultados

- La derivada se muestra en forma simbólica.  
- La integral es una **aproximación numérica**.  
- Se recomienda usar funciones continuas y sin singularidades en el intervalo definido.
